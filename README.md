# SW485-Project - Group 12

## Project Title
Obesity Risk Classification and Lifestyle Advice System

## Motivation
Obesity is one of the most significant global health challenges today.
This project analyzes lifestyle and health data to predict obesity risk
levels and provide personalized health advice using machine learning.

## Dataset
- Name: Obesity or CVD Risk Dataset
- Source: https://www.kaggle.com/datasets/aravindpcoder/obesity-or-cvd-risk-classifyregressorcluster
